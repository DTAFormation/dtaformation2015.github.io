Introduction

Plusieurs langages composent l'écosystème Java tel que Java, Groovy, Scala, Ruby, Javascript. Ce sont des langages portables sur la plupart des OS grâce à la machine virtuelle (JVM). C'est à dire que du bytecode est généré puis exécuté sur la JVM. La machine virtuelle d'Androïd est de très mauvaise qualité.
